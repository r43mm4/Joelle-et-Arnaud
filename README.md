## Wedding faire-part — Joëlle & Arnaud

Mini site statique (sans dépendances) pour un faire-part numérique.

### Structure
- `index.html` : page principale
- `styles.css` : styles
- `script.js` : interactions (RSVP mailto, ajout au calendrier, etc.)
- `assets/` : images (à ajouter)

### Ouvrir le site
Double-clique `index.html` (ou ouvre-le dans un navigateur).

### Personnaliser
Édite directement le contenu dans `index.html` (noms, date, lieu, programme, texte).
